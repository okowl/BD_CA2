This is a simple web application, that integretates 3 different API:

Twitter, OAUTH, Yandex MAP
1. In order for it's to work, please enter a twitter nickname into the field below.
2. Allow the popup with twitter authorisation.
3. Then it will automatically scan top 20 followers of the selected twitter user and place geotag mark on the map below.
4. If you have any issues to test it locally please, check the deployed version on github pages.
